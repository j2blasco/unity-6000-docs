### Parameters

value | The [InputDevice](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/XR.InputDevice.html) that just connected.  
### Description
Defines the delegate to use to register events when an [InputDevice](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/XR.InputDevice.html) is connected.
This delegate allows you to receive device connection events, so you know when the list of devices changes.
