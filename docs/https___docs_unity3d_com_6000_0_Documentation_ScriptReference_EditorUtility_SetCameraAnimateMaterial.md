## Declaration
public static void SetCameraAnimateMaterials([Camera](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Camera.html) camera, bool animate); 
### Description
Sets this camera to allow animation of materials in the Editor.
If enabled, any rendered objects take global time into consideration. This means that any custom previews or Scene view cameras can update in the Editor when not in Play Mode.  
  
Additional resources: [EditorUtility.SetCameraAnimateMaterialsTime](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/EditorUtility.SetCameraAnimateMaterialsTime.html).
