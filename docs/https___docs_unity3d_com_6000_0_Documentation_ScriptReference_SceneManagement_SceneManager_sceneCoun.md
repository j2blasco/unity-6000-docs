sceneCountInBuildSettings; 
### Description
Number of Scenes in Build Settings.
The number of Scenes which have been added to the [Build Settings](https://docs.unity3d.com/6000.0/Documentation/Manual/BuildSettings.html). The Editor will contain Scenes that were open before entering playmode.
