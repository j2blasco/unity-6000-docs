## Declaration
public void ResetWorldToCameraMatrix(); 
### Description
Make the rendering position reflect the camera's position in the Scene.
Call this to end the effect of setting [worldToCameraMatrix](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Camera-worldToCameraMatrix.html).
