[Vector2](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Vector2.html) minSize; 
### Description
The minimum size of this window when it is floating or modal. The minimum size is not used when the window is docked.
An EditorWindow can be resized. The [minSize](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/EditorWindow-minSize.html) variable provides the minimum width and height that is supported. This is typically a Vector2(100,100).
