**Obsolete** Training data export for baked lighting is deprecated. public string trainingDataDestination; 
### Description
Determines the name of the destination folder for the exported textures. (Editor only).
For example, "myProject/30samples" would create a folder called “30samples”in the root-level Project folder. To be able to export textures, see [exportTrainingData](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/LightingSettings-exportTrainingData.html).
