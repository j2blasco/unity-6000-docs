[Quaternion](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Quaternion.html) bodyRotation; 
### Description
The human body orientation for that pose.
Average body orientation. The average body orientation up vector is computed out of the hips and shoulders middle points. The front vector is then the cross product of the up vector and average left/right hips/shoulders vectors.
