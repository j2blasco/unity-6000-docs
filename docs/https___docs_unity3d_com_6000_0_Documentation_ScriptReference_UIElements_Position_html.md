### Description
Defines how the position values are interpreted by the layout engine. 
### Properties
Property | Description  
---|---  
[Relative](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/UIElements.Position.Relative.html) |  The element is positioned in relation to its default box as calculated by layout.   
[Absolute](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/UIElements.Position.Absolute.html) |  The element is positioned in relation to its parent box and doesn't contribute to the layout anymore.
