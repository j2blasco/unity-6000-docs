## Declaration
public static bool GetIgnoreLayerCollision(int layer1, int layer2); 
### Description
Are collisions between `layer1` and `layer2` being ignored?
Returns the value set by [Physics.IgnoreLayerCollision](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Physics.IgnoreLayerCollision.html) or in the Physics inspector.
