[RuntimeAnimatorController](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/RuntimeAnimatorController.html) runtimeAnimatorController; 
### Description
The Runtime Animator Controller that the Animator Override Controller overrides.
Note: Animator Override Controllers cannot be nested, which means you cannot supply an Animator Override Controller to [AnimatorOverrideController.runtimeAnimatorController](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/AnimatorOverrideController-runtimeAnimatorController.html).
