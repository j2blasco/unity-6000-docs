filteringGaussianRadiusIndirect; 
### Description
Specifies the method that the Progressive Lightmapper uses to reduce noise in lightmaps. (Editor only).
Options are `None`, `Automatic`, or `Advanced`.  
  
This setting affects lightmaps generated by the Baked Global Illumination system. It only applies to the the Progressive Lightmapper.  
  
When Unity serializes this `LightingSettings` object as a [Lighting Settings Asset](https://docs.unity3d.com/6000.0/Documentation/Manual/class-LightingSettings.html), this property corresponds to the **Filtering** property in the Lighting Settings Asset Inspector.  
  
Additional resources: [Lighting Settings Asset](https://docs.unity3d.com/6000.0/Documentation/Manual/class-LightingSettings.html).
