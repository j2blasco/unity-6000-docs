### Description
Memory statistics.
The detail pane (bottom half) of the [ProfilerWindow](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/ProfilerWindow.html) includes the ability to take detailed snapshots of type PackedMemorySnapshot via MemorySnapshot.RequestNewSnapshot and display it.
