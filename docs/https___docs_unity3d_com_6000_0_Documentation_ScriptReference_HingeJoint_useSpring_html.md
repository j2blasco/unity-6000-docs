useSpring; 
### Description
Enables the joint's [spring](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/HingeJoint-spring.html). Disabled by default.
Additional resources: [JointSpring](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/JointSpring.html)  
  
Enabling the [motor](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/HingeJoint-motor.html) **overrides** the spring, given the spring was enabled. If the motor is again disabled the spring will be restored.
