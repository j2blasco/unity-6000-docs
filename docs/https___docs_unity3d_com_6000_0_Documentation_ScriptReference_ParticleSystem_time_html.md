time; 
### Description
Playback position in seconds.
Use this to read the current playback time or to seek to a new playback time. For a looping system, this value wraps within the duration. For systems that specify a Start Delay, getting the time ignores the delay.  
  
Additional resources: ParticleSystem.MainModule.totalTime, [ParticleSystem.MainModule.startDelay](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/ParticleSystem.MainModule-startDelay.html).
