### Description
Automatically choose the render mode.
This chooses whether to render the [Light](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Light.html) as a pixel or vertex light (recommended and default). Additional resources: [light component](https://docs.unity3d.com/6000.0/Documentation/Manual/class-Light.html).
