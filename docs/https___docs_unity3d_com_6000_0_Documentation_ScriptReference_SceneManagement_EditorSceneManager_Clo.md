## Declaration
public static bool ClosePreviewScene([SceneManagement.Scene](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/SceneManagement.Scene.html) scene); 
### Parameters
Parameter | Description  
---|---  
scene | The preview Scene to close.  
### Returns
**bool** True if the Scene was successfully closed. 
### Description
Closes a preview Scene created by [NewPreviewScene](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/SceneManagement.EditorSceneManager.NewPreviewScene.html) or [OpenPreviewScene](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/SceneManagement.EditorSceneManager.OpenPreviewScene.html).
