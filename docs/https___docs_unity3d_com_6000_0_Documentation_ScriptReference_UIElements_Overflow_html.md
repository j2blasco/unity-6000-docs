### Description
Defines what should happened if content overflows an element bounds. 
### Properties
Property | Description  
---|---  
[Visible](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/UIElements.Overflow.Visible.html) |  The overflow isn't clipped. It renders outside the element's box. Default Value.   
[Hidden](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/UIElements.Overflow.Hidden.html) |  The overflow is clipped, and the rest of the content will be invisible.
