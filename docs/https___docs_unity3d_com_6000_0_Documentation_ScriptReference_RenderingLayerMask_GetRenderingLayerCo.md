## Declaration
public static int GetRenderingLayerCount(); 
### Returns
**int** The number of all existing Rendering Layers. 
### Description
Returns a number of Rendering Layers defined in the [Tags and Layers manager](https://docs.unity3d.com/6000.0/Documentation/Manual/class-TagManager.html) including empty ones.
