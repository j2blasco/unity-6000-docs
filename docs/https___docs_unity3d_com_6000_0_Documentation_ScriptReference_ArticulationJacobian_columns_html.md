### Description
Number of columns of the matrix is equal to the total number of all joint degrees of freedom(DOF), plus 6 if [ArticulationBody.immovable](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/ArticulationBody-immovable.html) is false.
