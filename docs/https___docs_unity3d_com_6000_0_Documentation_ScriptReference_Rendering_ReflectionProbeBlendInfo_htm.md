### Description
ReflectionProbeBlendInfo contains information required for blending probes.
Additional resources: [Renderer.reflectionProbeUsage](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Renderer-reflectionProbeUsage.html).
### Properties
Property | Description  
---|---  
[probe](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Rendering.ReflectionProbeBlendInfo-probe.html) | Reflection Probe used in blending.  
[weight](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Rendering.ReflectionProbeBlendInfo-weight.html) | Specifies the weight used in the interpolation between two probes, value varies from 0.0 to 1.0.
