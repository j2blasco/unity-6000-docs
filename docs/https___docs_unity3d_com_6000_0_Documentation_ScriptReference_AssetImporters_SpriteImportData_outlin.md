### Description
Sprite Asset creation uses this outline when it generates the Mesh for the [Sprite](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Sprite.html). If this is not given, SpriteImportData.tesselationDetail will be used to determine the mesh detail.
