isFocused; 
### Description
Whether the Player currently has focus (Read Only).
In the Editor, this corresponds to whether the Game view has focus (regardless of whether the Editor is in Play mode or not).  
  
Additional resources: [MonoBehaviour.OnApplicationFocus](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/MonoBehaviour.OnApplicationFocus.html).
