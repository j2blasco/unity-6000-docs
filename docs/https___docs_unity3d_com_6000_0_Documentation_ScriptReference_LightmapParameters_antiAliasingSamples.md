### Description
The kernel width the lightmapper uses when sampling a lightmap texel.
The kernel width the lightmapper uses when sampling a lightmap texel. This is the square root of [AntiAliasingSamples](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/LightmapParameters.AntiAliasingSamples.html).
