## Declaration
public static [AnimationCurve](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/AnimationCurve.html) Linear(float timeStart, float valueStart, float timeEnd, float valueEnd); 
### Parameters
Parameter | Description  
---|---  
timeStart | The start time for the linear curve.  
valueStart | The start value for the linear curve.  
timeEnd | The end time for the linear curve.  
valueEnd | The end value for the linear curve.  
### Returns
**AnimationCurve** The linear curve created from the specified values. 
### Description
A straight Line starting at `timeStart`, `valueStart` and ending at `timeEnd`, `valueEnd`.
