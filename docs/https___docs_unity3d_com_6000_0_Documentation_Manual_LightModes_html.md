# Set the Mode of a Light
To assign a **Mode** , select the Light in your **Scene** A Scene contains the environments and menus of your game. Think of each unique Scene file as a unique level. In each Scene, you place your environments, obstacles, and decorations, essentially designing and building your game in pieces. [More info](https://docs.unity3d.com/6000.0/Documentation/Manual/CreatingScenes.html)  
See in [Glossary](https://docs.unity3d.com/6000.0/Documentation/Manual/Glossary.html#Scene) and, in the [Light Inspector window](https://docs.unity3d.com/6000.0/Documentation/Manual/class-Light.html), select **Mode**.
