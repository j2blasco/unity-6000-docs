### Description
Handles UI for one shader property ignoring any custom drawers.
This is similar to [ShaderProperty](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/MaterialEditor.ShaderProperty.html) function, except it ignores any custom [MaterialPropertyDrawer](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/MaterialPropertyDrawer.html) objects.
