## Declaration
public [Hash128](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Hash128.html) GetValueFromBag([UIElements.IUxmlAttributes](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/UIElements.IUxmlAttributes.html) bag, [UIElements.CreationContext](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/UIElements.CreationContext.html) cc); 
### Parameters
Parameter | Description  
---|---  
bag | The bag of attributes.  
cc | The context in which the values are retrieved.  
### Returns
**Hash128** The value of the attribute. 
### Description
Retrieves the value of this attribute from the attribute bag. Returns the value if found; otherwise, it returns defaultValue.
