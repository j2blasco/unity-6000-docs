### Description
Defines the types of text generators to use.
This enum is used to switch between Unity's standard and advanced text generators.
### Properties
Property | Description  
---|---  
[Standard](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/TextGeneratorType.Standard.html) | The standard text generator, which is the default option.  
[Advanced](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/TextGeneratorType.Advanced.html) | Supports comprehensive Unicode and text shaping for various languages and scripts, including right-to-left (RTL) languages.
