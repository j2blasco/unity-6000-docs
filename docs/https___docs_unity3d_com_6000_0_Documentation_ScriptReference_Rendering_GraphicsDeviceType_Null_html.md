### Description
No graphics API.
This typically happens when a "null" graphics API is explicitly requested from command line arguments, for example when running game servers or editor in batch mode.  
  
Additional resources: [SystemInfo.graphicsDeviceType](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/SystemInfo-graphicsDeviceType.html).
