### Description
Identifies a specific revision for a [Git package](https://docs.unity3d.com/6000.0/Documentation/Manual/upm-git.html) using a Git commit hash.
### Properties
Property | Description  
---|---  
[hash](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/PackageManager.GitInfo-hash.html) | Returns the resolved Git commit hash for the requested revision for this package.  
[revision](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/PackageManager.GitInfo-revision.html) | Returns the requested Git revision for the Git package.
