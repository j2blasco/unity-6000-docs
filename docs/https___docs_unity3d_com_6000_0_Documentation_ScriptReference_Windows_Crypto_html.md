### Description
Class representing cryptography algorithms.
Note: this class is only available when targeting Universal Windows Platform.

Method | Description  
---|---  
[ComputeMD5Hash](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Windows.Crypto.ComputeMD5Hash.html) | Computes MD5 hash value for the specified byte array.
