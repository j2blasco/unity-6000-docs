streamingMipmapsMaxLevelReduction; 
### Description
The maximum number of mipmap levels to discard for each texture.
This is also used as the number of mipmaps to discard the first time the texture is loaded, before texture streaming begins to bring in the desired mipmap levels based on camera calculations.
