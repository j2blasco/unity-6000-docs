### Description
A collection of common color functions.
### Static Methods
Method | Description  
---|---  
[ToHtmlStringRGB](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/ColorUtility.ToHtmlStringRGB.html) | Returns the color as a hexadecimal string in the format "RRGGBB".  
[ToHtmlStringRGBA](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/ColorUtility.ToHtmlStringRGBA.html) | Returns the color as a hexadecimal string in the format "RRGGBBAA".
