## Declaration
public void Reload(); 
### Description
Call this to force the TreeView to reload its data. This in turn causes BuildRoot and BuildRows to be called.
Additional resources: [BuildRoot](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/IMGUI.Controls.TreeView.BuildRoot.html).
