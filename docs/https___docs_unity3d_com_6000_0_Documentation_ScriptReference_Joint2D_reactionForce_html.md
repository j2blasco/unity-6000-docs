[Vector2](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Vector2.html) reactionForce; 
### Description
Gets the reaction force of the joint.
When a joint tries to constrain a Rigidbody2D it may need to apply a force to do so. This is known as the reaction force.  
  
Additional resources: [GetReactionForce](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Joint2D.GetReactionForce.html), [breakForce](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Joint2D-breakForce.html).
