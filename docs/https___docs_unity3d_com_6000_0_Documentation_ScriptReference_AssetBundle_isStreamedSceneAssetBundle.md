isStreamedSceneAssetBundle; 
### Description
Return true if the AssetBundle contains Unity Scene files
An AssetBundle can store either Scenes or Assets, never a mix of the two. Additional resources: [AssetBundle.GetAllScenePaths](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/AssetBundle.GetAllScenePaths.html)
