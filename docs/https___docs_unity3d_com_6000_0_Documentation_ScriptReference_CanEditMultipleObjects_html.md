### Description
Attribute used to make a custom editor support multi-object editing.
When multiple objects are selected, Editors that don't have this attribute will show the message "Multi-object editing not supported". See the [Editor](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Editor.html) class documentation on how to make an Editor support multi-object editing.  
  
Additional resources: [Editor](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Editor.html) class.
