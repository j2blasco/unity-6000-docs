### Description
The available filter kernels for the Progressive Lightmapper.
### Properties
Property | Description  
---|---  
[Gaussian](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/LightingSettings.FilterType.Gaussian.html) | When enabled, the lightmap uses a Gaussian filter.  
[ATrous](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/LightingSettings.FilterType.ATrous.html) | When enabled, the lightmap uses an A-Trous filter.  
[None](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/LightingSettings.FilterType.None.html) | When enabled, the lightmap uses no filtering.
