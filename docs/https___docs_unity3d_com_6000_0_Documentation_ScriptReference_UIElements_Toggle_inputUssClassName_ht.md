inputUssClassName; 
### Description
USS class name of input elements in Toggle elements. 
Unity adds this USS class to the input sub-element of the [Toggle](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/UIElements.Toggle.html). The input sub-element provides responses to the manipulator.
