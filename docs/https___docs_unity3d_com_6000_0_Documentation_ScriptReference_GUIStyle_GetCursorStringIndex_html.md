## Declaration
public int GetCursorStringIndex([Rect](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Rect.html) position, [GUIContent](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/GUIContent.html) content, [Vector2](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Vector2.html) cursorPixelPosition); 
### Description
Get the cursor position (indexing into contents.text) when the user clicked at cursorPixelPosition.
This does not respect any images inside content.
