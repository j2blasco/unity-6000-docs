canGetSelection; 
### Description
Specifies whether the TouchScreenKeyboard supports the [selection](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/TouchScreenKeyboard-selection.html) property. (Read Only)
This property is supported on Android, iOS, and UWP only. On unsupported platforms, this property always returns false.
