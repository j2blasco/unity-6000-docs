recording; 
### Description
This property toggles recording in the Animation window.
Recording allows the Animation window to update the animation clip with modifications to the Game Object in the Scene View. Enabling this also enables [AnimationWindow.previewing](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/AnimationWindow-previewing.html).
