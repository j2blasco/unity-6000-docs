### Description
8-bit unsigned normalized number.
This format stores uniformly quantized numbers in [0..1] range, with 8 bits of precision (256 possible different values). For example, [Mesh.colors32](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Mesh-colors32.html) use this format.
