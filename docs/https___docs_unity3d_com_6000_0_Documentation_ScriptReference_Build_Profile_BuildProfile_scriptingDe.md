scriptingDefines; 
### Description
Lists the user-specified script compilation defines in the build profile.
Player and Editor scripts include these script compilation defines only if the build profile is active. Additional resources: [PlayerSettings](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/PlayerSettings.html).
