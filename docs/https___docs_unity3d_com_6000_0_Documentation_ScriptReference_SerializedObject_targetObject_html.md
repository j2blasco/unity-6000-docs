targetObject; 
### Description
The inspected object (Read Only).
When multi-object editing, the targetObject property will just return the first of the edited objects. To get all the objects, use the [targetObjects](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/SerializedObject-targetObjects.html) property instead.
