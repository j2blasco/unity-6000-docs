### Description
Mesh is a line strip.
First two indices form a line, and then each new index connects a new vertex to the existing line strip.  
  
Additional resources: [Mesh.SetIndices](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Mesh.SetIndices.html) function.
