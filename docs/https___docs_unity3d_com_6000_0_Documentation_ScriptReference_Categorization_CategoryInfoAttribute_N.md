### Description
The name to use for this category.
If Name is not specified, the element name will be used. This can falback on type name if no [ElementInfoAttribute](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Categorization.ElementInfoAttribute.html) specifying Name.
