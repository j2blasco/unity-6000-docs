### Description
Determines whether the mouse cursor is rendered using software rendering or, on supported platforms, hardware rendering.
See [Cursor](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Cursor.html) for supported platforms.
### Properties
Property | Description  
---|---  
[Auto](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/CursorMode.Auto.html) | Use hardware cursors on supported platforms.  
[ForceSoftware](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/CursorMode.ForceSoftware.html) | Force the use of software cursors.
