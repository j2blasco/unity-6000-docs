## Declaration
public void ApplyBuiltinRootMotion(); 
### Description
Apply the default Root Motion.
Applies the default root motion. Use this in [MonoBehaviour.OnAnimatorMove](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/MonoBehaviour.OnAnimatorMove.html) or in [StateMachineBehaviour.OnStateMove](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/StateMachineBehaviour.OnStateMove.html) on frames where you don't want to handle the root motion manually.
