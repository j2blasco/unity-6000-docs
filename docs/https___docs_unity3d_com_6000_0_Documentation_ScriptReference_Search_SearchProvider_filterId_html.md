filterId; 
### Description
Text token used to "filter" by search provider (ex: "me:", "p:", "s:").
Note that it is mandatory to use the filter for an explicit provider (see [SearchProvider.isExplicitProvider](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Search.SearchProvider-isExplicitProvider.html) for an example).
