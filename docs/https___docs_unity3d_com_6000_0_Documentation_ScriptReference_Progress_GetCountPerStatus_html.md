## Declaration
public static int[] GetCountPerStatus(); 
### Returns
**int[]** An array that contains the count of progress indicators per status. Each index maps a single status from the [Status](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Progress.Status.html) enum. 
### Description
For each available status, gets the number of progress indicators with that status.
