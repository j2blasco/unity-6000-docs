## Declaration
public static ChannelInfo[] GetChannelList(); 
### Returns
**ChannelInfo[]** A list that contains the ChannelInfo for every open channel. 
### Description
Gets a list of channels open in the ChannelService. By default the ChannelService always has a "status" channel and an "events" channel.
