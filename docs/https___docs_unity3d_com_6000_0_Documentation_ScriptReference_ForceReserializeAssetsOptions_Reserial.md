### Description
Specifies that [AssetDatabase.ForceReserializeAssets](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/AssetDatabase.ForceReserializeAssets.html) should load, upgrade, and save the .meta files for the assets at the paths passed to the function, but not the assets themselves.
