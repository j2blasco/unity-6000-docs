### Description
Constants for special values of [Screen.sleepTimeout](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Screen-sleepTimeout.html).
Use them to specify something other than a fixed amount of seconds before dimming the screen.
### Static Properties
Property | Description  
---|---  
[NeverSleep](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/SleepTimeout.NeverSleep.html) | Prevent screen dimming.  
[SystemSetting](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/SleepTimeout.SystemSetting.html) | Set the sleep timeout to whatever the user has specified in the system settings.
