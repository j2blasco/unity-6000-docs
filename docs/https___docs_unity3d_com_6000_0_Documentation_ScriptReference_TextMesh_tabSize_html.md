tabSize; 
### Description
How much space will be inserted for a tab '\t' character. This is a multiplum of the 'spacebar' character offset.
```
using UnityEngine;  
  
public class Example : MonoBehaviour[](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/MonoBehaviour.html)
{
    void Start()
    {
        GetComponent<TextMesh[](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/TextMesh.html)>().tabSize = 5;
    }
}

```
