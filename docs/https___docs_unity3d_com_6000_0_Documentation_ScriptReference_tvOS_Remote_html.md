### Description
A class for Apple TV remote input configuration.
### Static Properties
Property | Description  
---|---  
[allowExitToHome](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/tvOS.Remote-allowExitToHome.html) | Configures how Menu button behaves on Apple TV Remote.   
[allowRemoteRotation](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/tvOS.Remote-allowRemoteRotation.html) | Configures if Apple TV Remote should autorotate all the inputs when Remote is being held in horizontal orientation. Default is false.  
[reportAbsoluteDpadValues](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/tvOS.Remote-reportAbsoluteDpadValues.html) | Configures how touches are mapped to analog joystick axes in relative or absolute values.   
[touchesEnabled](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/tvOS.Remote-touchesEnabled.html) | Disables Apple TV Remote touch propagation to Unity Input.touches API.
