## Declaration
public float GetHeight(); 
### Description
Override this method to specify how tall the GUI for this decorator is in pixels.
The default is one line high. See [DecoratorDrawer](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/DecoratorDrawer.html) for an example of how to use this.
