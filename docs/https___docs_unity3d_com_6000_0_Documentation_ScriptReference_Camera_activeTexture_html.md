[RenderTexture](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/RenderTexture.html) activeTexture; 
### Description
Gets the temporary RenderTexture target for this Camera.
The current Camera being rendered applies its image to this temporary render target. The render target is accessible in the Camera's OnPostRender method.  
  
Additional resources: [Camera.OnPostRender](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Camera.OnPostRender.html)
