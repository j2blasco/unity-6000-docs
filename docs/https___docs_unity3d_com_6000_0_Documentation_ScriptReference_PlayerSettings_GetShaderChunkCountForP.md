## Declaration
public static int GetShaderChunkCountForPlatform([BuildTarget](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/BuildTarget.html) buildTarget); 
### Parameters
Parameter | Description  
---|---  
buildTarget | The build target to get the shader chunk count for.  
### Description
Gets the default limit on the number of shader variant chunks Unity loads and keeps in memory for the build target.
