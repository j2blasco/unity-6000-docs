pressure; 
### Description
The current amount of pressure being applied to a touch. 1.0f is considered to be the pressure of an average touch. If [Input.touchPressureSupported](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Input-touchPressureSupported.html) returns false, the value of this property will always be 1.0f.
