useDynamicScaleExplicit; 
### Description
When this flag is set to true, render texture is set to be used by the [Dynamic Resolution system](https://docs.unity3d.com/6000.0/Documentation/Manual/DynamicResolution.html). Scale is applied with an explicit call to [ApplyDynamicScale](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/RenderTexture.ApplyDynamicScale.html)
