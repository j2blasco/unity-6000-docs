footerAddButtonName; 
### Description
The name of the add button element in the footer. 
Unity uses this name of [BaseListView](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/UIElements.BaseListView.html) add button when showAddRemoveFooter is set to `true`.
