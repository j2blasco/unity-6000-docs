### Description
Use [JsonUtility.ToJson](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/JsonUtility.ToJson.html) to serialize a message argument. Additional metadata is added to the message to specify which type to convert the JSON to when it is received.
