### Description
Contains the geometric shape of a Terrain collider.
The only way to retrieve this shape is to use the TerrainCollider.Geometry property to get it from a [TerrainCollider](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/TerrainCollider.html) component.
### Properties
Property | Description  
---|---  
[GeometryType](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/LowLevelPhysics.TerrainGeometry.GeometryType.html) | Returns the geometry type of this shape, which is TerrainGeometry.
