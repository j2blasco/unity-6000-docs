## Declaration
public void Sleep(); 
### Description
Forces an articulation body to sleep.
A common use is to call this from Awake in order to make an articulation body sleep at startup. See the [Rigidbodies Overview](https://docs.unity3d.com/6000.0/Documentation/Manual/RigidbodiesOverview.html) in the manual for more information about Rigidbody/ArticulationBody sleeping.
