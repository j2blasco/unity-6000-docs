### Description
Choose how Particle Trails are generated.
### Properties
Property | Description  
---|---  
[PerParticle](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/ParticleSystemTrailMode.PerParticle.html) | Makes a trail behind each particle as the particle moves.  
[Ribbon](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/ParticleSystemTrailMode.Ribbon.html) | Draws a line between each particle, connecting the youngest particle to the oldest.
