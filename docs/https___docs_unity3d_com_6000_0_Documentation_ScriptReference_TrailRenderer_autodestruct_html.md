autodestruct; 
### Description
Does the [GameObject](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/GameObject.html) of this Trail Renderer auto destruct?
When set to `true`, the [GameObject](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/GameObject.html) will be destroyed when all its points have been removed.
