### Description
Options for which application entries to include when Unity generates a Gradle project.
### Properties
Property | Description  
---|---  
[Activity](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/AndroidApplicationEntry.Activity.html) | Include an application entry which derives from the Activity class.  
[GameActivity](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/AndroidApplicationEntry.GameActivity.html) | Include an application entry which derives from the GameActivity class.
