### Description
The type of the current process. It can be a Unity master instance, or a secondary instance connected to the master.
### Properties
Property | Description  
---|---  
[Main](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/MPE.ProcessLevel.Main.html) | The current process is a main instance of UnityEditor.  
[Secondary](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/MPE.ProcessLevel.Secondary.html) | The current process is a secondary instance of UnityEditor that is connected to a master instance.
