## Declaration
public static [AudioConfiguration](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/AudioConfiguration.html) GetConfiguration(); 
### Returns
**AudioConfiguration** The new configuration to be applied. 
### Description
Returns the current configuration of the audio device and system. The values in the struct may then be modified and reapplied via [AudioSettings.Reset](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/AudioSettings.Reset.html).
See [AudioSettings.Reset](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/AudioSettings.Reset.html) for an example.
