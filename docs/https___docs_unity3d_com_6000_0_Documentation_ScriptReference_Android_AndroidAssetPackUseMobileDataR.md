### Description
Represents the choice of an end user that indicates if your application can use mobile data to download Android asset packs.
The value is received from a callback after AndroidAssetPacks.RequestToUseMobileDataAsync is called. Additional resources: [AndroidAssetPacks.RequestToUseMobileDataAsync](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Android.AndroidAssetPacks.RequestToUseMobileDataAsync.html).
