### Description
Options to control the Unity References to other assembly definition files that Unity uses during compilation.
Additional resources: AssemblyBuilder.referencesOptions.
### Properties
Property | Description  
---|---  
[None](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Compilation.ReferencesOptions.None.html) | Default options.  
[UseEngineModules](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Compilation.ReferencesOptions.UseEngineModules.html) | Use UnityEngine modules when compiling.
