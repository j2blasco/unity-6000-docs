gridResolutionX; 
### Description
The 3D grid resolution on the x-axis.
This property is used only when the [resolutionMode](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/LightProbeProxyVolume-resolutionMode.html) is set to **Custom**. The final resolution is the closest power of 2.
