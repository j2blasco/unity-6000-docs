### Description
Provides an interface to the system that draws the Unity trial version and development build watermarks in screen space.
### Static Methods
Method | Description  
---|---  
[IsVisible](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Rendering.Watermark.IsVisible.html) | Returns true if the watermark is visible on-screen. Whether the watermark is visible depends on your build settings and Unity plan.
