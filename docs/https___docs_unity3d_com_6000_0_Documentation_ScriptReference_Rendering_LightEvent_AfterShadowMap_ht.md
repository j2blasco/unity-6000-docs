### Description
After shadowmap is rendered.
This value indicates all shadow casters are rendered and the current render target is still the shadow map. Note that shadow cascade parameters are not yet set. See [LightEvent.BeforeScreenspaceMask](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Rendering.LightEvent.BeforeScreenspaceMask.html) for more information.
