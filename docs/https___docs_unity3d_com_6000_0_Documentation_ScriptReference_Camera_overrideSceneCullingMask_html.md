### Description
Sets the culling mask used to determine which objects from which Scenes to draw. See [EditorSceneManager.SetSceneCullingMask](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/SceneManagement.EditorSceneManager.SetSceneCullingMask.html).
