[Texture2D](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Texture2D.html) grayTexture; 
### Description
Gets a small Texture with all gray pixels.
Unity sets all pixels of this Texture to gray (0.5,0.5,0.5,0.5). The alpha value is also 0.5 (half transparent), and the color is in sRGB color space.  
  
Additional resources: [linearGrayTexture](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Texture2D-linearGrayTexture.html).
