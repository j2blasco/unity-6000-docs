spatializePostEffects; 
### Description
Determines if the spatializer effect is inserted before or after the effect filters.
This flag only has an effect if the spatialize flag is enabled on the AudioSource. See [AudioSource.spatialize](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/AudioSource-spatialize.html) for further information about spatialization.
