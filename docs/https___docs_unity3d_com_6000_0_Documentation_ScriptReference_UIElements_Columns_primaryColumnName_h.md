primaryColumnName; 
### Description
Indicates the column that needs to be considered as the primary column, by ID. 
Needs to match a [Column](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/UIElements.Column.html)'s id, otherwise will be ignored. The primary column cannot be hidden and will contain the expand arrow for tree views.
