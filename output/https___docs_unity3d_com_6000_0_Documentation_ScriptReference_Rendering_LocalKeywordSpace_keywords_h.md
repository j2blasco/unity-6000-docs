keywords; 
### Description
An array containing all [LocalKeyword](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Rendering.LocalKeyword.html) structs in this local keyword space. (Read Only)
Use this property to retrieve all local shader keywrods in this `LocalKeywordSpace`.
