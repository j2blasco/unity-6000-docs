newParentInstanceId; 
### Description
The instance ID of the [GameObject](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/GameObject.html) that is the new parent of the target. Note that this is not the instance ID of its [Transform](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Transform.html).
