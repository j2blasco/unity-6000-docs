# Dedicated Server requirements
You must meet the following requirements to use the Dedicated Server build target.
  1. Download and install Unity Editor version **2021.3 LTS** or later.
  2. Install the Dedicated Server module for the platform youâ€™re planning to build your application on. For example, enable **Linux Dedicated Server Build Support** if you plan to build for Linux.
