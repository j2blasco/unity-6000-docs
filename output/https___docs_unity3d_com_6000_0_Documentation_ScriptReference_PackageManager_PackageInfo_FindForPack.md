## Declaration
public static PackageInfo FindForPackageName(string name); 
### Parameters
Parameter | Description  
---|---  
name | The package name to look for.  
### Returns
**PackageInfo** The [PackageInfo](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/PackageManager.PackageInfo.html) instance describing the package, or null if there is no registered package with the given name. 
### Description
Retrieves information about the package based on the name of that package.
