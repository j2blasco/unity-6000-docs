[PlayerSettings.SplashScreen.DrawMode](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/PlayerSettings.SplashScreen.DrawMode.html) drawMode; 
### Description
Determines how the Unity logo should be drawn, if it is enabled. If no Unity logo exists in [logos] then the draw mode defaults to [DrawMode.UnityLogoBelow](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/PlayerSettings.SplashScreen.DrawMode.UnityLogoBelow.html).
