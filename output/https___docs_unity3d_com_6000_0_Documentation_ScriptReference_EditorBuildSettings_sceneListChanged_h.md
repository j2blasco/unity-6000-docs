### Description
A delegate called whenever [EditorBuildSettings.scenes](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/EditorBuildSettings-scenes.html) is set.
Used by any system that needs to be aware of updates to the Scenes list. Is only called if the Scenes array actually changes.
