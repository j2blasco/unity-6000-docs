### Description
Indicates when the search index was last modified.
In example, this timestamp is used with the [SearchMonitor.GetDiff](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Search.SearchMonitor.GetDiff.html) function to gather all the changes that happened since the last time this index was modified.
