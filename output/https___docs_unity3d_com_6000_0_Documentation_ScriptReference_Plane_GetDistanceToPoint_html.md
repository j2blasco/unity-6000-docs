## Declaration
public float GetDistanceToPoint([Vector3](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Vector3.html) point); 
### Description
Returns a signed distance from plane to point.
The value returned is positive if the point is on the side of the plane into which the plane's normal is facing, and negative otherwise.
