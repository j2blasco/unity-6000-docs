## Declaration
public static Object InstanceIDToObject(int instanceID); 
### Parameters
Parameter | Description  
---|---  
instanceID | Instance ID of an Object.  
### Returns
**Object** Resolved reference or null if the instance ID didn't match anything. 
### Description
Translates an instance ID to an object reference.
