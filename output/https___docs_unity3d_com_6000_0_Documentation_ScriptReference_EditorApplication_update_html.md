[EditorApplication.CallbackFunction](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/EditorApplication.CallbackFunction.html) update; 
### Description
Delegate for generic updates.
Add your functions to this delegate to get an update.  
  
This delegate is not called at a fixed rate. Rather, it is called at least once, but usually multiple times per seconds.  
  
Additional resources: [CallbackFunction](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/EditorApplication.CallbackFunction.html).
