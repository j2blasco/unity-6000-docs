### Parameters
Parameter | Description  
---|---  
value | Unity calls this delegate when the tracking boundary changes.  
### Description
An event that takes the delegate instance that the [XRInputSubsystem](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/XR.XRInputSubsystem.html) calls when it changes its tracking boundary.
See XRInputSubsystem.GetBoundaryPoints to get the new boundary.
