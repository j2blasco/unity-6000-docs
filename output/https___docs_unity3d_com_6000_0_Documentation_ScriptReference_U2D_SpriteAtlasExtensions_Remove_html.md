## Declaration
public static void Remove([U2D.SpriteAtlas](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/U2D.SpriteAtlas.html) spriteAtlas, Object[] objects); 
### Parameters
Parameter | Description  
---|---  
objects | Object in the array you wish to remove.  
### Description
Remove objects from the atlas's packable objects list.
If there are multiple of the exact same Object stored inside the atlas, all copies will be removed at once.  
  
Additional resources: [SpriteAtlasExtensions.Add](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/U2D.SpriteAtlasExtensions.Add.html).
