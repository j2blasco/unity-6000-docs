### Description
Wrapping modes for text that reaches the horizontal boundary.
### Properties
Property | Description  
---|---  
[Wrap](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/HorizontalWrapMode.Wrap.html) | Text will word-wrap when reaching the horizontal boundary.  
[Overflow](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/HorizontalWrapMode.Overflow.html) | Text can exceed the horizontal boundary.
