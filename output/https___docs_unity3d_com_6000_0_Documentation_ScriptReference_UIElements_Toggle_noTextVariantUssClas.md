noTextVariantUssClassName; 
### Description
USS class name of Toggle elements that have no text. 
Unity adds this USS class to the [Toggle](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/UIElements.Toggle.html) if the Toggle does not have a label.
