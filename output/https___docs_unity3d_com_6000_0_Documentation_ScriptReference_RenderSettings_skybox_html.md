### Description
The global skybox to use.
If you change the skybox in playmode, you have to use the [DynamicGI.UpdateEnvironment](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/DynamicGI.UpdateEnvironment.html) function call to update the ambient probe.
