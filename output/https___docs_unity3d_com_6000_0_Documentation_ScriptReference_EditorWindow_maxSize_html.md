[Vector2](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Vector2.html) maxSize; 
### Description
The maximum size of this window when it is floating or modal. The maximum size is not used when the window is docked.
An EditorWindow can be resized. The [maxSize](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/EditorWindow-maxSize.html) variable provides the largest width and height that is supported. This is typically a Vector2(4000,4000). Note that this size is likely to be larger than most screen resolutions.
