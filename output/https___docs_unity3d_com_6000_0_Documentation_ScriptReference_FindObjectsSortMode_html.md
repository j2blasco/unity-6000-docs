### Description
Options to specify if and how to sort objects returned by a function.
### Properties
Property | Description  
---|---  
[None](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/FindObjectsSortMode.None.html) | Don't sort the objects.  
[InstanceID](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/FindObjectsSortMode.InstanceID.html) | Sort the objects by InstanceID in ascending order.
