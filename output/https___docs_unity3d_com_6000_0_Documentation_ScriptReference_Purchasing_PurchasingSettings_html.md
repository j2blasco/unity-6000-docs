### Description
Editor API for the Unity Services editor feature. Normally Purchasing is enabled from the Services window, but if writing your own editor extension, this API can be used.
### Static Properties
Property | Description  
---|---  
[enabled](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Purchasing.PurchasingSettings-enabled.html) | This Boolean field will cause the Purchasing feature in Unity to be enabled if true, or disabled if false.
