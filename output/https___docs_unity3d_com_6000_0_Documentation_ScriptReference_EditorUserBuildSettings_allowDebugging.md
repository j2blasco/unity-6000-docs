allowDebugging; 
### Description
Enable source-level debuggers to connect.
You can also set this property in the Unity Editor using the [Script Debugging](https://docs.unity3d.com/6000.0/Documentation/Manual/build-profiles-reference.html#shared-build-settings) setting on the **Build Profiles** window.
