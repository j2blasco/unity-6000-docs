### Description
The limits defined by the [CharacterJoint](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/CharacterJoint.html).
### Properties
Property | Description  
---|---  
[bounciness](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/SoftJointLimit-bounciness.html) | When the joint hits the limit, it can be made to bounce off it.  
[contactDistance](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/SoftJointLimit-contactDistance.html) | Determines how far ahead in space the solver can "see" the joint limit.  
[limit](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/SoftJointLimit-limit.html) | The limit position/angle of the joint (in degrees).
