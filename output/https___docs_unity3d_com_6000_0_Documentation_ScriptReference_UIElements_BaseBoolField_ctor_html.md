## Declaration
public BaseBoolField(string label); 
### Parameters
Parameter | Description  
---|---  
label | The Label text.  
### Description
Creates a [BaseBoolField](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/UIElements.BaseBoolField.html) with a Label and a default manipulator. 
The default manipulator makes it possible to activate the BaseBoolField with a left mouse click.
