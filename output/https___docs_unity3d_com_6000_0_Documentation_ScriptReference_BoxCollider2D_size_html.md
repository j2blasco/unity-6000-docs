[Vector2](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Vector2.html) size; 
### Description
The width and height of the rectangle.
The X coordinate of the vector represents the width while the Y represents the height. These values are specified relative to a center point, so the distance from the center to the left edge is actually width/2.  
  
Additional resources: [center](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/BoxCollider2D-center.html).
