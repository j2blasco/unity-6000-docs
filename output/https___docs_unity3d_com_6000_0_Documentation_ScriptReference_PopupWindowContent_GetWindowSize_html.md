## Declaration
public [Vector2](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Vector2.html) GetWindowSize(); 
### Returns
**Vector2** The size of the Popup window. 
### Description
The size of the popup window.
Implement this method only when not using [PopupWindowContent.CreateGUI](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/PopupWindowContent.CreateGUI.html) to provide the content of the popup.
