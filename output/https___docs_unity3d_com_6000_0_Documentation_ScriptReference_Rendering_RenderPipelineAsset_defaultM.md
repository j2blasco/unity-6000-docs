[Material](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Material.html) defaultMaterial; 
### Returns
**Material** Default material. 
### Description
Return the default [Material](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Material.html) for this pipeline.
This is used whenever a 3D object is created in the Editor.  
  
Note: This material is also applied to imported submeshes with 0 triangles.
