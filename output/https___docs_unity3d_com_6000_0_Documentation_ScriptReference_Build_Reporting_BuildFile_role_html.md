### Description
The role the file plays in the build output.
Use this field to understand what purpose a file serves within the built player. Common roles for files are captured by the members of the [CommonRoles](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Build.Reporting.CommonRoles.html) class.
