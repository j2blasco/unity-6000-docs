### Description
Encapsulates a Texture2D and its shader property name to give Sprite-based renderers access to a secondary texture, in addition to the main Sprite texture.
### Properties
Property | Description  
---|---  
[name](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/SecondarySpriteTexture-name.html) | The shader property name of the secondary Sprite texture. Use this name to identify and sample the texture in the shader.  
[texture](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/SecondarySpriteTexture-texture.html) | The texture to be used as a secondary Sprite texture.
