shadowBias; 
### Description
Shadow mapping constant bias.
Shadow caster surfaces are pushed by this world-space amount away from the light, to help prevent self-shadowing ("shadow acne") artifacts.  
  
Additional resources: [shadows](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Light-shadows.html), [shadowNormalBias](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Light-shadowNormalBias.html) properties.
