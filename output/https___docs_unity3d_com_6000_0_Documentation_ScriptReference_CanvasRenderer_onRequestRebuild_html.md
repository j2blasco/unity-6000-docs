### Description
(Editor Only) Event that gets fired whenever the data in the [CanvasRenderer](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/CanvasRenderer.html) gets invalidated and needs to be rebuilt.
For instance, whenever a Texture gets re-imported this event gets fired.
