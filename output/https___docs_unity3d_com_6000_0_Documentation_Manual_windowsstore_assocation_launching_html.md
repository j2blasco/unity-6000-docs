# Association launching for UWP
You can use the association launching API to launch the userâ€™s default app for a file type or protocol. For more information, refer to [`UnityEngine.WSA.Launcher`](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/WSA.Launcher.html). You can also enable your app to be the default app for a file type or protocol in [**Player settings**](https://docs.unity3d.com/6000.0/Documentation/Manual/class-PlayerSettingsWSA.html) > **File and Protocol Associations**. More settings are available in `Package.appxmanifest` > **Declarations**.
