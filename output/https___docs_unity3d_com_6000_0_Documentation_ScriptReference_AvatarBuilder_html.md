### Description
Class to build avatars from user scripts.
This class allows you to create custom avatars for your animated characters entirely via script, in a similar way to what goes on behind the Scenes in the Unity Editor when you create an avatar from the model import inspector.
### Static Methods
Method | Description  
---|---  
[BuildGenericAvatar](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/AvatarBuilder.BuildGenericAvatar.html) | Create a new generic avatar.  
[BuildHumanAvatar](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/AvatarBuilder.BuildHumanAvatar.html) | Create a humanoid avatar.
