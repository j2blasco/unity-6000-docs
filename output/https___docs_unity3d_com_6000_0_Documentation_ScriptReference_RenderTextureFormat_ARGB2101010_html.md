### Description
Color render texture format. 10 bits for colors, 2 bits for alpha.
Note that not all graphics cards support this format. Use [SystemInfo.SupportsRenderTextureFormat](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/SystemInfo.SupportsRenderTextureFormat.html) to check for support.  
  
Additional resources: [RenderTexture](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/RenderTexture.html) class, [SystemInfo.SupportsRenderTextureFormat](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/SystemInfo.SupportsRenderTextureFormat.html).
