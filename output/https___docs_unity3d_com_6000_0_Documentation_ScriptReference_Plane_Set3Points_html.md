## Declaration
public void Set3Points([Vector3](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Vector3.html) a, [Vector3](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Vector3.html) b, [Vector3](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Vector3.html) c); 
### Parameters
Parameter | Description  
---|---  
a | First point in clockwise order.  
b | Second point in clockwise order.  
c | Third point in clockwise order.  
### Description
Sets a plane using three points that lie within it. The points go around clockwise as you look down on the top surface of the plane.
Additional resources: [SetNormalAndPosition](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Plane.SetNormalAndPosition.html).
