[Gradient](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Gradient.html) strokeGradient; 
### Description
The stroke gradient to use when using Stroke. 
Setting a stroke gradient will override the currently set strokeColor. Setting a null stroke gradient will remove it and fall back on the currently set strokeColor.
