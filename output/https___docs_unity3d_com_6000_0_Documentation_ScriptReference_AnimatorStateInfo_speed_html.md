speed; 
### Description
The playback speed of the animation. 1 is the normal playback speed.
A negative playback speed will play the animation from the end.  
  
see [AnimatorState.speed](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Animations.AnimatorState-speed.html).
