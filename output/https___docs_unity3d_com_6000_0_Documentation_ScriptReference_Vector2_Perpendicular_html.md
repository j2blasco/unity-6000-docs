## Declaration
public static [Vector2](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Vector2.html) Perpendicular([Vector2](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Vector2.html) inDirection); 
### Parameters
Parameter | Description  
---|---  
inDirection | The input direction.  
### Returns
**Vector2** The perpendicular direction. 
### Description
Returns the 2D vector perpendicular to this 2D vector. The result is always rotated 90-degrees in a counter-clockwise direction for a 2D coordinate system where the positive Y axis goes up.
