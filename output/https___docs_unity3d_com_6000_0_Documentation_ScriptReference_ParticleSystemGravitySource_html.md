### Description
Options for which physics system to use the gravity setting from.
### Properties
Property | Description  
---|---  
[Physics3D](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/ParticleSystemGravitySource.Physics3D.html) | Use gravity from the 3D physics system.  
[Physics2D](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/ParticleSystemGravitySource.Physics2D.html) | Use gravity from the 2D physics system.
