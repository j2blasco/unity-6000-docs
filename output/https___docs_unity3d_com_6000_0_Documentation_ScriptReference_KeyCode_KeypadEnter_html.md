### Description
Numeric keypad `Enter`.
This is a key at the far right of a keyboard. Some laptops do not have a numeric keypad.  
  
The `Enter` key is documented as [KeyCode.Return](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/KeyCode.Return.html).
