# Edit Collider mode reference
The actions and shortcuts available when **Edit**Collider** An invisible shape that is used to handle physical collisions for an object. A collider doesn’t need to be exactly the same shape as the object’s mesh - a rough approximation is often more efficient and indistinguishable in gameplay. [More info](https://docs.unity3d.com/6000.0/Documentation/Manual/CollidersOverview.html)  
See in [Glossary](https://docs.unity3d.com/6000.0/Documentation/Manual/Glossary.html#Collider)** is enabled.
Action | Function  
---|---  
**Click and drag vertex or edge** | Move the selected vertex and edge to a different position.  
**Click anywhere along an edge** | Create a new vertex by clicking an empty space along the edge.  
**Hold the Ctrl(macOS:Cmd) key and select a vertex or edge** | Deletes the selected vertex or edge.
