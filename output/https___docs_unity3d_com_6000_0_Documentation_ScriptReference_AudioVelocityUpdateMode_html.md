### Description
Describes when an [AudioSource](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/AudioSource.html) or [AudioListener](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/AudioListener.html) is updated.
### Properties
Property | Description  
---|---  
[Auto](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/AudioVelocityUpdateMode.Auto.html) | Updates the source or listener in the FixedUpdate loop if it is attached to a Rigidbody, dynamic Update otherwise.  
[Fixed](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/AudioVelocityUpdateMode.Fixed.html) | Updates the source or listener in the FixedUpdate loop.  
[Dynamic](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/AudioVelocityUpdateMode.Dynamic.html) | Updates the source or listener in the dynamic Update loop.
