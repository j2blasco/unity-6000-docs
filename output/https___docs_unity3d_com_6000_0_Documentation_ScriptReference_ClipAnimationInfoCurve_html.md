### Description
Stores a curve and its name that will be used to create additional curves during the import process.
### Properties
Property | Description  
---|---  
[curve](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/ClipAnimationInfoCurve-curve.html) | The animation curve.  
[name](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/ClipAnimationInfoCurve-name.html) | The name of the animation curve.
