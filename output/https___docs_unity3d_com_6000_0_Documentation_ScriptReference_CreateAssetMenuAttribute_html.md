### Description
Mark a ScriptableObject-derived type to be automatically listed in the Assets/Create submenu, so that instances of the type can be easily created and stored in the project as ".asset" files.
### Properties
Property | Description  
---|---  
[fileName](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/CreateAssetMenuAttribute-fileName.html) | The default file name used by newly created instances of this type.  
[menuName](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/CreateAssetMenuAttribute-menuName.html) | The display name for this type shown in the Assets/Create menu.  
[order](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/CreateAssetMenuAttribute-order.html) | The position of the menu item within the Assets/Create menu.
