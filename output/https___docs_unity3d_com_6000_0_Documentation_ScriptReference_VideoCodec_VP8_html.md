### Description
Encode video using the vp8 codec.
The VP8 video codec is supported by software on most platforms, and by hardware on a few platforms such as Android and Web. For more information about video encoding, refer to [Video encoding compatibility reference](https://docs.unity3d.com/6000.0/Documentation/Manual/video-encoding-compatibility.html).
