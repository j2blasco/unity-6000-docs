[Color](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Color.html) contactSeparationColor; 
### Description
Contact point separation color.
The color of the line that represents the [ContactPoint.separation](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/ContactPoint-separation.html) distance. This color is also used by the disc that shows where the separation ends and the [ContactPoint.point](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/ContactPoint-point.html) is.
