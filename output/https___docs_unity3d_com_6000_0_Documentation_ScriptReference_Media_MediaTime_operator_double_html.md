### Description
Returns the time value expressed as a floating point number of seconds.
A loss of precision may result when performing the conversion. Converting invalid time values (see [isValid](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Media.MediaTime-isValid.html)) returns 0.
