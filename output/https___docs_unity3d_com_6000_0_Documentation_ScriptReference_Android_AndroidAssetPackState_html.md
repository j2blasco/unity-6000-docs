### Description
Represents the state of a single Android asset pack.
The state is received as a callback after AndroidAssetPacks.GetAssetPackStateAsync is called. Additional resources: [AndroidAssetPacks.GetAssetPackStateAsync](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Android.AndroidAssetPacks.GetAssetPackStateAsync.html).
### Properties
Property | Description  
---|---  
[error](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Android.AndroidAssetPackState-error.html) | Indicates an error code that describes what happened when querying the Android asset pack state.  
[name](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Android.AndroidAssetPackState-name.html) | The name of the Android asset pack the status query is for.  
[status](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Android.AndroidAssetPackState-status.html) | The status of the Android asset pack.
