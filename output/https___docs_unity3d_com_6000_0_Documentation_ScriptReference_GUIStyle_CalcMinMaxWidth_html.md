## Declaration
public void CalcMinMaxWidth([GUIContent](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/GUIContent.html) content, out float minWidth, out float maxWidth); 
### Description
Calculate the minimum and maximum widths for this style rendered with `content`.
Used by [GUILayout](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/GUILayout.html) to handle word-wrapping elements correctly.
