### Description
A receiver is required for SendMessage.
If no receiver is found, an error is printed to the console. (This is the default value.) Additional resources: [GameObject.SendMessage](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/GameObject.SendMessage.html)
