## Declaration
public static [U2D.SpriteAtlasAsset](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/U2D.SpriteAtlasAsset.html) Load(string assetPath); 
### Parameters
Parameter | Description  
---|---  
assetPath | The path of the SpriteAtlasAsset file on disk.  
### Returns
**SpriteAtlasAsset** Returns the loaded SpriteAtlasAsset. 
### Description
Loads SpriteAtlasAsset at the given path. File extension of SpriteAtlasAsset is *.spriteatlasv2.
