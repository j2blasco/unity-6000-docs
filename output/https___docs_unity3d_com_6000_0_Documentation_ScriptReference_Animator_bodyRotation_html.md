### Description
The rotation of the body center of mass.
The rotation is in worldspace. You should only set this value from within an [OnAnimatorIK()](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/MonoBehaviour.OnAnimatorIK.html) function call.
