### Description
An extension to the [BuildReport](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Build.Reporting.BuildReport.html) class that tracks how Assets contribute to the size of the build.
The build process generates a PackedAssets report for each .sharedAssets or .resource file, or for each AssetBundle.
### Properties
Property | Description  
---|---  
[contents](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Build.Reporting.PackedAssets-contents.html) | An array of PackedAssetInfo objects that holds information about the Assets that are included in the PackedAssets bundle, such as packed Asset size and type.  
[overhead](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Build.Reporting.PackedAssets-overhead.html) | The header size of the packed Asset file.  
[shortPath](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Build.Reporting.PackedAssets-shortPath.html) | The file path to the Asset package, relative to the Data folder of the build.
