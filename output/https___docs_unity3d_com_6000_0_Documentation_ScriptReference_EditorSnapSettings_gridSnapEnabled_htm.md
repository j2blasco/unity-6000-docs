gridSnapEnabled; 
### Description
Gets or sets whether grid snapping is enabled.
When enabled, grid snapping will snap the translate handle to the nearest point on the [SceneView](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/SceneView.html) grid when the Tool Handle Rotation is set to "Global".
