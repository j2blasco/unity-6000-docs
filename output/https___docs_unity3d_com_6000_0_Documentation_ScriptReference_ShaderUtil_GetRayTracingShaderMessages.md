## Declaration
public static ShaderMessage[] GetRayTracingShaderMessages([Rendering.RayTracingShader](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Rendering.RayTracingShader.html) s); 
### Parameters
Parameter | Description  
---|---  
s | The [RayTracingShader](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Rendering.RayTracingShader.html) instance to check for messages.  
### Returns
**ShaderMessage[]** An array of [ShaderMessage](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/ShaderMessage.html) structs containing the generated messages. 
### Description
Returns each error and warning generated by the Shader Compiler for the given [RayTracingShader](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Rendering.RayTracingShader.html).
