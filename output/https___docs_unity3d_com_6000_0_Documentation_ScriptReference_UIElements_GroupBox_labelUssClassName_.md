labelUssClassName; 
### Description
USS class name for Labels in GroupBox elements. 
Unity adds this USS class to the [Label](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/UIElements.Label.html) sub-element of the [GroupBox](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/UIElements.GroupBox.html) if the GroupBox has a Label.
