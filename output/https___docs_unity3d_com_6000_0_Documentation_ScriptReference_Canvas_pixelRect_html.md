[Rect](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Rect.html) pixelRect; 
### Description
Get the render rect for the [Canvas](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Canvas.html).
If in overlay mode this will be the screen dimensions. If in world mode this will be the camera screen viewport rect.
