### Description
The width of the trail at the end of the trail.
A width of 1 corresponds to 1 unit in the game world.  
  
Additional resources: [startWidth](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/TrailRenderer-startWidth.html) variable.
