instance; 
### Description
Gets the instance of the Singleton. Unity creates the Singleton instance when this property is accessed for the first time. If you use the [FilePathAttribute](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/FilePathAttribute.html), then Unity loads the data on the first access as well.
