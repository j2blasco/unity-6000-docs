offsetFactor; 
### Description
Scales the maximum Z slope in the GPU's depth bias setting.
Corresponds to the `Offset` command in ShaderLab. For more information, see [ShaderLab command: Offset](https://docs.unity3d.com/6000.0/Documentation/Manual/SL-Offset.html).
