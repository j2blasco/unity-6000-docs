[Vector3](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Vector3.html) currentTorque; 
### Description
The torque applied by the solver to satisfy all constraints.
The returned value is relative to [Joint.connectedBody](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Joint-connectedBody.html) if it's set. Otherwise, in world space. Additional resources: [Joint.currentForce](https://docs.unity3d.com/6000.0/Documentation/ScriptReference/Joint-currentForce.html).
